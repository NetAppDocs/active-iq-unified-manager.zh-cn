= 配置事件保留设置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以指定事件在自动删除之前在 Unified Manager 服务器中保留的月数。

* 您需要的内容 *

您必须具有应用程序管理员角色。

将事件保留 6 个月以上可能会影响服务器性能，因此不建议这样做。

.步骤
. 在左侧导航窗格中，单击 * 常规 * > * 数据保留 * 。
. 在 * 数据保留 * 页面中，选择事件保留区域中的滑块并将其移动到事件应保留的月数，然后单击 * 保存 * 。
