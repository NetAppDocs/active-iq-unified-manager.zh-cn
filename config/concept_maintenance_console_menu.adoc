= 维护控制台菜单
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
维护控制台包含多个不同的菜单，可用于维护和管理 Unified Manager 服务器的特殊功能和配置设置。

根据安装 Unified Manager 的操作系统，维护控制台包含以下菜单：

* 升级 Unified Manager （仅限 VMware ）
* 网络配置（仅限 VMware ）
* 系统配置（仅限 VMware ）
+
.. 支持/诊断
.. 重置服务器证书
.. 外部数据提供程序
.. 备份还原
.. 性能轮询间隔配置
.. 禁用 SAML 身份验证
.. 查看/更改应用程序端口
.. 调试日志配置
.. 控制对MySQL端口3306的访问
.. 退出




您可以从列表中选择用于访问特定菜单选项的编号。例如、对于备份和还原、请选择_4_。
