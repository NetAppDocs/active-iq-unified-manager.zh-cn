= 查看用户
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用用户页面查看使用 Unified Manager 管理存储对象和数据的用户列表。您可以查看有关用户的详细信息，例如用户名，用户类型，电子邮件地址以及分配给用户的角色。

* 您需要的内容 *

您必须具有应用程序管理员角色。

.步骤
. 在左侧导航窗格中，单击 * 常规 * > * 用户 * 。
