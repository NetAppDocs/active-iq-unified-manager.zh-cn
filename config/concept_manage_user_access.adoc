= 管理用户访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以创建角色并分配功能来控制用户对Active IQ Unified Manager 的访问。您可以确定具有访问Unified Manager中选定对象所需功能的用户。只有具有这些角色和功能的用户才能在Unified Manager中管理对象。
