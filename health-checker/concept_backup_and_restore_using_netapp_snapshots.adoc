= 使用 NetApp Snapshot 进行备份和还原
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Snapshot 备份会为 Unified Manager 数据库和配置文件创建一个时间点映像，您可以使用此映像在发生系统故障或数据丢失时进行还原。您计划定期将 Snapshot 备份写入其中一个 ONTAP 集群上的卷，以便始终具有最新副本。

[NOTE]
====
此功能不适用于虚拟设备上安装的 Active IQ Unified Manager 。

====
