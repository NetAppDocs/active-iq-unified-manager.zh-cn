= 未找到用户
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在执行添加，编辑，删除或测试远程用户或组等与身份验证相关的操作时，将显示以下错误消息： `User not found.`

* 发生原因 *

如果用户位于 AD 服务器或 LDAP 服务器中，并且您将基本可分辨名称设置为不正确的值，则可能会发生此问题。

* 更正操作 *

. 在左侧导航窗格中，单击 * 常规 * > * 远程身份验证 * 。
. 为基本可分辨名称输入相应的信息。
. 单击 * 保存 * 。
