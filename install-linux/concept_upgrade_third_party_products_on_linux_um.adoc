= 安装 Unified Manager 后升级第三方产品
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果 Unified Manager 已安装在 Linux 系统上，则可以升级第三方产品，例如 JRE 。

开发这些第三方产品的公司会定期报告安全漏洞。您可以按自己的计划升级到此软件的较新版本。
