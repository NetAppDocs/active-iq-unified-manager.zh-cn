= 通过信息板监控集群性能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager 信息板提供了几个面板，用于显示此 Unified Manager 实例所监控的所有集群的性能状态概况。通过它，您可以评估受管集群的整体性能，并快速记录，查找或分配任何已确定的特定事件以解决问题。
