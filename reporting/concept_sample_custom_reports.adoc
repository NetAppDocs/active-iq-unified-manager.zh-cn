= 示例自定义报告
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
这些示例自定义报告通常用于帮助您确定潜在问题并在出现问题之前对潜在问题做出响应。

本节中的报告列表并不详尽，并且会随着时间的推移而增长。您可以通过提供文档反馈来建议将自定义报告添加到此部分。

[NOTE]
====
您必须具有应用程序管理员或存储管理员角色才能管理报告。

====
