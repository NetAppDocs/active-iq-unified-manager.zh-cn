= 对数据进行排序
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要对结果进行排序，请单击一列并指示升序或降序。对数据排序会优先考虑报告所需的信息。

.步骤
. 单击列顶部对结果进行排序，以便最重要的信息显示在视图顶部。
. 为未保存的视图命名以保存所做的更改。
